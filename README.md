cookiecutter-raml
=================

Cookiecutter template for creating a RAML 1.0 project.

Requirements
------------
Install `cookiecutter` command line: `pip install cookiecutter`    

Usage
-----
Generate a new Cookiecutter template layout: `cookiecutter gh:genzj/cookiecutter-raml`
